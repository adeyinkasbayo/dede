QUICK DEPLOYMENT STEPS:
================================================================================

1. EXTRACT THE ZIP FILE
   - Extract darfiden_management_system.zip
   - You will get a "public_html" folder with all files

2. UPLOAD TO YOUR HOSTING
   - Upload contents of "public_html" folder to your web server
   - Common locations: /public_html/, /www/, /httpdocs/

3. CREATE MYSQL DATABASE
   - Login to cPanel or phpMyAdmin
   - Create a new database (e.g., "darfiden_db")
   - Create a database user with strong password
   - Grant ALL PRIVILEGES to the user

4. IMPORT DATABASE SCHEMA
   - In phpMyAdmin, select your database
   - Click "Import" tab
   - Upload: sql/darfiden_full_schema.sql
   - Click "Go" to import

5. CONFIGURE DATABASE CONNECTION
   - Edit file: src/config.php
   - Update these lines:
     define('DB_HOST', 'localhost');      // Your DB host
     define('DB_NAME', 'darfiden_db');    // Your DB name
     define('DB_USER', 'your_username');  // Your DB username
     define('DB_PASS', 'your_password');  // Your DB password

6. SET FOLDER PERMISSIONS
   - Set permissions for uploads folder:
     chmod 755 uploads/
     chmod 755 uploads/passports/
     chmod 755 uploads/winnings/

7. CREATE ADMIN ACCOUNT
   - Visit: http://yourdomain.com/create_admin.php
   - You'll see success message
   - Default credentials created: admin / admin123

8. SECURE THE SYSTEM
   - DELETE create_admin.php file immediately
   - Login and change admin password
   - Enable HTTPS (uncomment lines in .htaccess)

COMMON ISSUES & SOLUTIONS:
================================================================================

Issue: "Database connection failed"

Solution: Ensure .htaccess is uploaded and mod_rewrite is enabled

Issue: ".htaccess causes 500 error"
Solution: Contact hosting provider to enable mod_rewrite
